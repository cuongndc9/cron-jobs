# cron-jobs

🚀💤 Node.js Cron Jobs.

## Quick start

```sh
# Install dependencies
$ yarn
# Run app
$ yarn start
```

*Need to allow non secure apps to access gmail you can do this by going to your gmail settings [here](https://myaccount.google.com/lesssecureapps).*
